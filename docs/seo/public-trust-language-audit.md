# Public Trust Language Audit

Audit date: 2026-07-10
Mode: audit snapshot with a later user-approved decision update
Scope: current public landing page, quiz framing, public result/offer, footer/FAQ, and public metadata on `codex/seo-geo-positioning`

## Decision Update

After reviewing the snapshot, the user approved these narrow copy changes:

- Positive product uses of `Diagnose` become `Auswertung`; medical boundary statements such as
  `keine Diagnose` remain unchanged.
- Homepage, FAQ, quiz-processing, offer, guarantee, and social-image claims use more specific,
  non-medical language and qualify uncertain outcomes.
- The four-week concept remains as a possible scenario or review point rather than a guaranteed
  result.

The user explicitly accepted the countdown and its shorter `Danach zum regulären Preis` line
unchanged for this rollout. The table below stays as the original audit snapshot.

## Priority

- **P0:** resolve or explicitly accept before the affected public surface ships.
- **P1:** include in the first user-approved trust-copy pass.
- **P2:** address in the next trust pass; material, but less likely to create immediate harm.

The file and line references below are an exact snapshot of the branch at the audit date. Suggested replacements are proposals only.
