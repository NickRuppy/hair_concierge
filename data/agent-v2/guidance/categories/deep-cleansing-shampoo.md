# Deep Cleansing Shampoo

## Purpose
Deep cleansing shampoo is an occasional reset for buildup, heavy styling residue, or over-conditioned feel.

## Use When
Use when the user reports buildup, greasy film, limp roots from residue, or needs a reset step.

## Agent May Decide
Explain whether a reset is needed before adding more care products.

## Code And Tools Decide
Concrete product IDs and cleansing claims come from select_products.

## Required Grounding
Use tool data for cleansing intensity and fit.

## Missing Required Data
Ask about styling/product buildup if it changes the reset decision.

## Constraint Conflicts
Do not recommend frequent aggressive cleansing for sensitive or irritated scalp.

## German Answer Shape
Describe it as occasional, follow with conditioner, and avoid overuse.

## Do Not
Do not make it a daily shampoo default.
