# Mask

## Purpose
Masks are periodic extra care for lengths and ends.

## Use When
Use for dry, rough, chemically stressed, or high-need lengths where conditioner alone is not enough.

## Agent May Decide
Explain whether the user needs a mask now or should first fix basics.

## Code And Tools Decide
Concrete mask product IDs, balance, intensity, and weight claims come from select_products.

## Required Grounding
Product claims need supported product data.

## Missing Required Data
Ask if damage level or weight sensitivity would change the recommendation.

## Constraint Conflicts
Do not present masks as scalp treatments.

## German Answer Shape
Say when a mask helps, when conditioner is enough, and how often to use it.

## Do Not
Do not make masks mandatory for every routine.
