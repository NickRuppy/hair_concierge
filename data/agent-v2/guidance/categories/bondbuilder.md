# Bondbuilder

## Purpose
Bondbuilders are targeted repair-lane products for chemically stressed, breaking, or structurally compromised hair.

## Use When
Use for bleaching, color, chemical treatments, snapping, severe breakage, or explicit bondbuilder questions.

## Agent May Decide
Explain whether bondbuilding is a priority, optional course, or not the best lever.

## Code And Tools Decide
Concrete lanes, protocols, lifecycle, and product IDs come from select_products.

## Required Grounding
Use supported claims for repair lane and protocol.

## Missing Required Data
Ask if chemical history or breakage severity would change the lane.

## Constraint Conflicts
Do not replace baseline conditioning with bondbuilder logic.

## German Answer Shape
State whether the user has a clear bondbuilder driver, then explain the lane.

## Do Not
Do not call bondbuilders normal moisture masks.
