# Minimal Routine

Use when:
- relevant memory says the user prefers a simple, minimal, or easy routine
- or the user asks for fewer steps, low effort, low maintenance, budget restraint, travel simplicity, or "only what I really need"

Advisor interpretation:
- Keep the plan as small as possible while still addressing the core problem.
- Prefer removing, reducing, or combining redundant steps before adding complexity.
- Optional extras must stay clearly optional.
- A minimal routine can still be effective; do not make it sound like a compromise unless there is a real tradeoff.

Category implications:
- Shampoo: Choose the scalp-appropriate core cleanser; avoid adding specialty shampoos unless the problem clearly needs one.
- Conditioner: Usually the core length-support step when lengths need slip, softness, or detangling.
- Leave-in: Optional and strongest when it solves a specific gap such as heat protection, tangles, or dry ends in one step.
- Mask: Occasional add-on only; not part of the default minimal baseline.
- Oil: Optional tiny finishing step only when it clearly helps ends or frizz without adding weight.
- Bondbuilder / repair: Optional targeted course when damage signals justify it; not a normal minimal-routine requirement.

Routine implications:
- Present the smallest viable routine first, then one optional upgrade path only if it materially improves the user's stated goal.
- Prefer dual-role products or cadence adjustments over extra daily steps.

Avoid:
- Do not list every possible extra step.
- Do not make optional products sound mandatory.
- Do not solve a minimal-routine request with a full category lecture.
- Do not remove needed scalp or length care just to make the routine look short.

Ask only if:
- the missing detail would change the minimum viable routine, such as scalp priority versus length damage priority.

Proactive next step:
- Offer a "minimum version" and a single "upgrade if needed" version.
