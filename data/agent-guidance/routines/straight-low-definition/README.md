# Straight And Low-Definition Wavy Routine

Use this routine framework when assembling or adjusting a full routine for:
- `hair_texture = straight`
- `hair_texture = wavy` when the user does not have strong curl-definition goals

Do not use it for curly or coily routines, strong wave/curl definition goals, medical scalp disease, severe structural damage before stabilization, or one-product styling requests.

Recommended load order:

1. `core-fit.md`
2. `assembly-rules.md`
3. `guardrails.md`
4. `followups.md` only if the agent needs one targeted question before assembling the routine

This runtime framework is distilled from `docs/human-summaries/straight-hair-routine.md` and is loaded through `routine:straight_low_definition`.
