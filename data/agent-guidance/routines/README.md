# Routine Guidance

This folder holds runtime-safe routine frameworks for the bounded-agent experiment.

Use routine guidance for hair-type-specific routine assembly, such as straight hair, low-definition wavy hair, curly hair, or coily hair.

Do not use this folder for:
- generic answer behavior, which belongs in `playbooks/`
- profile modifiers, which belong in `overlays/`
- domain knowledge topics, which belong in `topics/`

Each routine framework can use:
- `README.md`
- `core-fit.md`
- `assembly-rules.md`
- `guardrails.md`
- `followups.md`

These files are groundwork only until registered in the bounded-agent loader.
