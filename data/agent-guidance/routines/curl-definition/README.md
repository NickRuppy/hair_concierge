# Curl Definition Routine

Use this routine framework when assembling or adjusting a full routine for:
- `hair_texture = curly`
- `hair_texture = coily`
- `hair_texture = wavy` when the user wants the wave pattern to show

Do not use it for straight hair, low-definition wavy routines, users who explicitly do not want wet styling, active medical scalp disease, or severe structural damage before stabilization.

Recommended load order:

1. `core-fit.md`
2. `assembly-rules.md`
3. `guardrails.md`
4. `followups.md` only if one targeted question is needed before assembling the routine

This runtime framework is distilled from `docs/human-summaries/curly-coily-routine.md` and is loaded through `routine:curl_definition`.
