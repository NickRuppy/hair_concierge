# CWC & OWC: Response Playbook

Use this module once CWC or OWC is already a plausible topic.

Default answer shape:
- explain first that this is a wash technique, not a special treatment
- name the likely direction: CWC for lighter protection, OWC for heavier protection
- tie the recommendation to the user's actual mismatch such as oily roots plus dry lengths, frequent washing, color fade, or dry detangling-prone lengths
- keep the benefit framing practical: gentler wash days, less roughness, less stripping
- ask at most one follow-up if it would change whether CWC or OWC is the better fit

Good reasons to ask one follow-up:
- it is unclear whether the user is closer to fine straight-wavy hair or coarse curly-coily hair
- it is unclear whether the main problem is dry lengths or an unhealthy scalp
- it is unclear whether the user already struggles with heavy residue or limp hair after washing

Routine or usage placement:
- frame both techniques as a way to run normal wash days more gently
- keep cadence flexible: this is a how-to-wash option, not a fixed schedule
- mention shampoo choice when color retention is a major goal, because technique helps but shampoo formulation still matters
- for OWC, keep the oil-selection advice lightweight and defer detailed oil choice to the hair-oiling topic if needed

Useful answer-safe phrasing:
- "This is less a treatment and more a gentler wash method for dry lengths."
- "CWC is usually the lighter starting point, while OWC is the heavier, more protective version."
- "If your scalp is the main problem, this should not replace scalp-focused care."

Keep deeper evidence detail, exact oil selection, and long special-case nuance out of the default runtime topic set.
