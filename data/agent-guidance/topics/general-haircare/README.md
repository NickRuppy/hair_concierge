# General Haircare

Use this topic for broad haircare questions that span washing, conditioning, drying, detangling, heat, accessories, environment, expectations, or basic scalp boundaries.

Do not use this as the first topic when the user is clearly asking about a narrower prepared topic such as hair oiling, CWC/OWC, deep cleansing, or bond builders.

Recommended load order:

1. `core-fit.md`
2. `response-playbook.md`
3. `guardrails.md`
4. `confusions.md` only if the user is mixing general care with myths, medical scalp issues, hair-loss claims, or a narrower topic

This runtime topic is distilled from `docs/human-summaries/general-haircare-advice.md` and is intentionally shorter than the human source.
